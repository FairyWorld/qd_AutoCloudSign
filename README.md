# 百度贴吧云签到

使用 PHP 编写的百度贴吧签到工具  
可能写的比较渣... QAQ  

## Demo
**由于更新 Demo 很麻烦，所以在线 Demo 暂时还是旧版本！**  
http://qwq233.tk/demos/cloudsign/

## 使用

把这个脚本放在任意一个 PHP 空间/VPS 上  
使用 IFTTT（如果是虚拟主机且不支持定时的话）或者任意其他的方法定时执行这个脚本  

访问首页，填写 Cooike 保存
